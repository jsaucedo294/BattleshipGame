#**Battleship Game** - Code Louisville

The Battleship Game is a console game that simulates the boardgame called Battleship. This game allows to play manually by entering the coordinance of the shots or automatically allowing you to play against your enemy with random shots. Also the games  keeps track of the players and their scores. 

## Installation
1. Open Visual Studio
2. Clone Repository
`copy and paste: https://github.com/jsaucedo294/BattleshipGame.git`
3. Open Solution
4. Run Application
`Press F5`
6. Play Battleship
